## 🎨 PART 1: Images Generate Karo

### Step 1 — File Open Karo
- `image-generator.html` ko computer / phone par download karo
- Browser mein open karo (Chrome / Safari)
- Page khulte hi **aapke saare 71 products** already pre-filled honge!

### Step 2 — Style Choose Karo
Top mein 6 customization options hain:

| Option | Choices |
|---|---|
| **Background** | Green Gradient (brand) / White / Cream / Dark / Colorful |
| **Image Size** | 512 × 512 / 800 × 800 / 1024 × 1024 / 600 × 400 banner |
| **Emoji Size** | Big / Medium / Small |
| **Brand Watermark** | Show / Hide |
| **Fresh Badge** | Show / Hide |
| **Product Name** | Show / Sirf Emoji |

**Mera Suggestion:**
- Background: **Green Gradient** (aapki brand color)
- Size: **800 × 800** (HD quality)
- Emoji Size: **Medium**
- Sab badges ON karo

### Step 3 — Download Karo
- Niche scroll karo — saare 71 cards preview mein dikhenge
- **"📦 Download ALL (ZIP)"** button dabao
- Ek `hariyalibasket-images.zip` file download hogi
- ZIP extract karo — 71 PNG images mil jayengi

### Step 4 — WordPress Mein Upload
1. WordPress Admin login
2. **Media → Add New**
3. Saari 71 PNG files drag-and-drop kar do
4. 30 second mein sab upload!
5. Product edit karte time **Featured Image** se select kar lo

## 📊 PART 2: Products CSV Import (Sabse Aasan Tarika!)

Aapke saare 71 products ek CSV mein hain — ek-ek manually banane ki zaroorat nahi.

### Step 1 — CSV Download Karo
GitHub se `products-master-list.csv` download karo:
👉 https://github.com/imanohar7773/freshdaily/blob/add-invoice-generator/products-master-list.csv

### Step 2 — WooCommerce Import
1. WordPress Admin login
2. **Products → All Products → Import** (top button)
3. Upload kiya hua `products-master-list.csv` choose karo
4. **Continue** dabao
5. WooCommerce automatically columns map kar dega
6. **Run the importer** dabao
7. **2 minute mein 71 products live!** 🎉

### CSV Mein Kya-Kya Hai?

Har product ke liye:
- ✅ **SKU** — unique code (HB-FR-001 format)
- ✅ **Name** — product naam
- ✅ **Categories** — Fruits / Root Vegetables / Green Vegetables / Herbs & Leafy / Exotic
- ✅ **Regular Price (MRP)** — example prices, aap edit kar sakte ho
- ✅ **Sale Price** — discounted price
- ✅ **Stock** — in stock by default
- ✅ **Short Description** — quick summary
- ✅ **Long Description** — detailed Hinglish description
- ✅ **Unit** — per Kg / per Pc / per Packet
- ✅ **Emoji** — har product ka emoji
- ✅ **Tax status** — taxable

### Prices Edit Karna Hai?

CSV ko **Excel** ya **Google Sheets** mein kholo:
1. `Regular price` aur `Sale price` columns dhundo
2. Apne actual prices update karo
3. Save karo (CSV format hi rakhna)
4. Fir import karo

## 💡 BONUS TIPS

### 1. Image Banner Bhi Bana Sakte Ho
Image generator mein **"600 × 400 (Banner)"** size choose karo — homepage banners ke liye perfect.

### 2. Sirf Kuch Products Generate Karne Hain?
Textarea mein extra products delete kar do — sirf jo chahiye unhi ke images banengi.

### 3. Images Quality Improve Karna Hai?
Size **1024 × 1024** choose karo — high-resolution images banengi.

### 4. CSV Mein Apne Variants Add Karne Hain (250g, 500g, 1Kg)?
Bata dena — main aapko **variant CSV** alag se bana doonga jisme har product ke 3 sizes ho.
