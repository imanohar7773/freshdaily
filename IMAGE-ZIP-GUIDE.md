## 🚀 USE KARNE KE 4 STEP:

### **STEP 1 — Tool Open Karo**

Direct browser mein kholo:
👉 https://raw.githack.com/imanohar7773/freshdaily/add-invoice-generator/hb-image-zip-builder.html

Ya download karke kholo:
1. Upar wala link kholo
2. Top right par **"Download raw file"** dabao
3. Browser mein open karo (Chrome/Safari)

### **STEP 2 — Images Verify Karo**

- Saari 71 product images preview mein dikhengi
- Koi image accha nahi laga? Card par 2 buttons:
  - **🔄 Change** — Pexels/Unsplash mein search → naya URL paste
  - **📤 Upload** — apni khud ki photo phone se upload (max 10MB)

### **STEP 3 — "📦 Download All as ZIP" Dabao**

- Top par bada orange button hai: **"📦 Download All as ZIP"**
- Click karo
- ⏳ 30-60 second wait — saari images fetch ho rahin
- Progress bar live dikhega
- Auto-download: `hariyalibasket-images.zip`

### **STEP 4 — WordPress Mein Upload**

1. ZIP extract karo (right-click → Extract)
2. 71 images mil jayengi, naam: `HB-FR-001-apple-green-imp.jpg` jaise
3. WordPress Admin → **Media → Add New**
4. Saari images drag & drop kar do (bulk upload)
5. ✅ Done! Ab har product mein **Featured Image** se select kar lo

## ❓ TROUBLESHOOTING

### "Some images failed"
- Internet slow ho sakta hai — re-try karo
- ya us specific image ka URL change karo manually

### "ZIP file blank/empty"
- JSZip library load nahi hui — internet check karo, refresh karo

### "Image preview hi nahi dikha rahi"
- URL galat ya image hat gayi — replace karo manual se

### "ZIP bahut bada hai" (50MB+)
- Normal hai — 71 high-quality images hain
- Internet WiFi pe download karo (mobile data save karo)
